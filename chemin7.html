Vous avez choisi le chemin 7, vous arrivez au

DONJON

Vous pénétrez dans l’ombre du donjon, où les torches vacillantes projettent des ombres inquiétantes sur les murs de pierre couverts de mousse. Malgré l’aura lugubre du lieu, une étrange énergie magique flotte dans l’air, illuminant les gravures anciennes qui ornent les piliers.

La porte étant entrouverte, vous descendez lentement l’escalier en colimaçon, chaque marche soulevant un nuage de poussière oubliée, mais un bruit soudain vous fige : les gardes de Kirby viennent d’entrer à l’étage supérieur, leurs voix résonnant à travers les couloirs de pierre.

Vous vous précipitez afin de trouver la sortie avant de vous faire attraper mais malheur ! Une impasse.


Vous doutez, il n'y a qu'un panneau sur lequel il est écrit :
"Vous risquez de vous tromper. Prenez à droite":
