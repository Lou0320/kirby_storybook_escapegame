# Livre Interactif - Kirby | Interactive Story - Kirby

*[English version below](#english-version)*

## Version Française

### Description
Ce projet est un **livre interactif en ligne** basé sur le concept "Un livre dont vous êtes le héros", avec pour thème l'univers magique de **Kirby**. Le joueur navigue dans l'histoire en faisant des choix qui influencent le déroulement du récit. **L'objectif : s'échapper de l'histoire !**

### Contexte du Projet
**Projet universitaire** réalisé dans le cadre du cours de **développement web (S4)**, axé sur la conception et le design de sites web interactifs.

### L'Univers Kirby
Kirby est un personnage emblématique de Nintendo, créé par Masahiro Sakurai et apparu pour la première fois en 1992 dans *Kirby's Dream Land* sur Game Boy. C'est une petite créature rose et ronde originaire de la planète Popstar, dotée d'une capacité unique : il peut aspirer ses ennemis et copier leurs pouvoirs. Son univers magique et coloré, ainsi que son gameplay accessible, en font une icône du jeu vidéo parfaite pour cette aventure interactive.

### Technologies Utilisées
- **HTML5** - Structure et contenu des pages
- **CSS3** - Stylisation et mise en page
- **Google Fonts** - Typographie personnalisée (Josefin Sans)

### Fonctionnalités
- **9 chemins différents** à explorer (chemin0 à chemin8)
- **Navigation interactive** entre les différentes pages
- **Interface utilisateur intuitive** avec barre de navigation
- **Responsive design** adapté aux différentes tailles d'écran
- **Éléments multimédias** (images, sons)
- **Conditions de fin** : victoire ou défaite selon vos choix

### Installation et Configuration

#### Prérequis
- Un navigateur web moderne (Chrome, Firefox, Safari, Edge)
- Aucune installation supplémentaire requise !

#### Comment lancer le projet
1. **Téléchargez** ou **clonez** ce repository
   ```bash
   git clone [URL_DU_REPOSITORY]
   ```
2. **Naviguez** vers le dossier du projet
3. **Ouvrez** le fichier `entree.html` dans votre navigateur web
4. **Commencez** votre aventure !

#### Structure du projet
```
kirby/
├── entree.html          # Page d'entrée
├── chemin0-8.html       # 9 chemins différents
├── win.html            # Page de victoire
├── loser.html          # Page de défaite
├── *.css               # Feuilles de style correspondantes
├── images/             # Ressources graphiques
├── son/               # Fichiers audio
└── README.md          # Ce fichier
```

### Comment Jouer
1. **Lisez** les consignes affichées sur la page d'entrée
2. **Faites des choix** en cliquant sur les options proposées
3. **Naviguez** dans l'histoire selon vos décisions
4. **Tentez de vous échapper** de l'histoire pour gagner !

## English Version <a id="english-version"></a>

### Description
This project is an **online interactive story book** based on the "Choose Your Own Adventure" concept, themed around the magical universe of **Kirby**. Players navigate through the story by making choices that influence the narrative flow. **The goal: escape from the story!**

### Project Context
**University project** developed as part of a **web development course (4th semester)**, focused on interactive website design and development.

### The Kirby Universe
Kirby is an iconic Nintendo character created by Masahiro Sakurai, first appearing in 1992 in *Kirby's Dream Land* for Game Boy. He's a small, pink, round creature from planet Popstar with a unique ability: he can inhale enemies and copy their powers. His magical and colorful universe, along with accessible gameplay, makes him a perfect video game icon for this interactive adventure.

### Technologies Used
- **HTML5** - Page structure and content
- **CSS3** - Styling and layout
- **Google Fonts** - Custom typography (Josefin Sans)

### Features
- **9 different paths** to explore (chemin0 to chemin8)
- **Interactive navigation** between different pages
- **Intuitive user interface** with navigation bar
- **Responsive design** adapted to different screen sizes
- **Multimedia elements** (images, sounds)
- **End conditions**: victory or defeat based on your choices

### Setup and Installation

#### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No additional installation required!

#### How to run the project
1. **Download** or **clone** this repository
   ```bash
   git clone [REPOSITORY_URL]
   ```
2. **Navigate** to the project folder
3. **Open** the `entree.html` file in your web browser
4. **Start** your adventure!

#### Project Structure
```
kirby/
├── entree.html          # Entry page
├── chemin0-8.html       # 9 different paths
├── win.html            # Victory page
├── loser.html          # Defeat page
├── *.css               # Corresponding stylesheets
├── images/             # Graphic resources
├── son/               # Audio files
└── README.md          # This file
```

### How to Play
1. **Read** the instructions displayed on the entry page
2. **Make choices** by clicking on the proposed options
3. **Navigate** through the story according to your decisions
4. **Try to escape** from the story to win!
